## Формулы расчёта

### ARPU (Average Revenue Per User)
```
ARPU = Всего доход за период / Количество пользователей за период
Пример: $10,000 / 1,000 пользователей = $10 ARPU
```

### ARPPU (Average Revenue Per Paying User)
```
ARPPU = Всего доход за период / Количество платящих пользователей за период
Пример: $10,000 / 100 платящих = $100 ARPPU
```

### LTV (Lifetime Value)
```
LTV = ARPU × Средняя длительность жизни пользователя (месяцы)
или
LTV = (ARPPU × Средний период подписки) / (1 - Retention Rate)
```

### CAC (Customer Acquisition Cost)
```
CAC = Все маркетинг затраты за период / Новые платящие юзеры за период
Пример: $1,000 на маркетинг / 50 новых юзеров = $20 CAC
```

### CAC Payback Period
```
CAC Payback = CAC / Средний ежемесячный доход с юзера
Пример: $20 CAC / $2 ARPU = 10 месяцев

```

### Churn Rate (Monthly)
```
Monthly Churn = (Пользователей в начале - Пользователей в конце + Новые) / Пользователей в начале
```

### Retention (Cohort-based)
```
Retention = Пользователи вернувшиеся в День N / Пользователи в День 0
Пример: 70% D7 retention = 70% пользователей вернулись на 7-й день
```
